## Using this library:
If you are using Maven, specify this library in your pom.xml file.
```
  <dependencies>
    <dependency>
    	<groupId>com.github.easai.math</groupId>
    	<artifactId>Complex</artifactId>
    	<version>0.0.1</version>
    </dependency>
  </dependencies>
```
